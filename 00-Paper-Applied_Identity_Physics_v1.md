## §3 · What Counts as an Applied IP Deposit

Not every applied writeup is an Applied IP corpus deposit. This section defines the standard.

**A deposit qualifies as Applied IP when it satisfies all five criteria:**

## §5 · The Applied IP Workflow

Every Applied IP deposit follows the same workflow. This section documents the workflow so future deposits can conform without ad hoc invention.

**Step 1 — Identify the substrate.** Name the real-world domain the deposit addresses. A specific composition class. A specific workforce structure. A specific meeting cadence. A specific clinical presentation. Applied IP does not reduce abstract categories; it reduces specific substrates.

**Step 2 — Cite the corpus dependency chain.** State which Foundational IP claims the deposit inherits (at minimum Ω₀ and TL) and which prior Applied IP deposits the substrate connects to (if any).

**Step 3 — Produce the PNBA map for the substrate.** Assign P, N, B, A values to the identities the substrate contains. For quantitative deposits, values are numerical. For qualitative deposits (early exploration), values are ranked or bounded.

**Step 4 — Apply the LDP six steps.** State the equation for the substrate. State known answers. Show the PNBA reduction. Name the operators. Show the work explicitly. Verify against known answers.

**Step 5 — Compute τ and classify the phase state.** Every identity in the deposit receives a τ = B/P value and a phase classification (Noble, Locked, or Shatter). This is the deposit's primary structural claim about the substrate.

**Step 6 — Identify the intervention or the structural implication.** If the phase state is Shatter, the deposit prescribes the intervention required to return to Locked. If the phase state is Locked or Noble, the deposit documents the structural implications the phase state enables.

**Step 7 — Formal verification (where applicable).** Produce a Lean 4 file at 0 sorry recording the deposit's core claims. The Lean file is the machine-checkable record; the paper is the human-readable reduction.

**Step 8 — Deposit under DOI and coordinate.** File the paper and its Lean file under the corpus DOI 10.5281/zenodo.18719748 with a corpus coordinate that locates the deposit within the Applied IP branch.

Deposits that follow the eight steps join the corpus. Deposits that do not are still valuable as applied work but are not corpus deposits and cannot be cited as such.
