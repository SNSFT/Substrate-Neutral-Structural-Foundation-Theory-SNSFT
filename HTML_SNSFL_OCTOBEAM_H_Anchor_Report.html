H-ANCHOR PRINCIPLE [9,9,2,3] · 8-BEAM

Hydrogen (H) is fixed as Beam 1 across all chaos agent collisions. Beams 2–8 are drawn randomly from the full PNBA corpus. This systematically maps the coupling geometry of H with all 7-element combinations reachable in 8-beam space.

When multiple different B2–B8 combinations produce the same output state (same phase, IM, B_out), the output is a structural attractor — physically real, not a sampling artifact. The variable beams are spectators. The invariant core is the signal. This is how CHON, FeS, and LiN were identified in the 4-beam parent run.

H IN 8-BEAM SPACE B=1 COUPLING CLASS

H (B=1) is the lowest non-Noble B in the corpus. In 8-beam space, H contributes k=min(1, B_partner) to each of its 7 pairwise couplings — always bottlenecked at k=1 per pair with any B≥1 partner. Total H contribution to k_max8: at most 7.

Noble condition: B_sum ≤ 2×k_max8. With 28 pairs and B_sum carrying contributions from 7 variable beams, the Noble fraction is structurally high. 99% Noble rate observed — consistent with 8-beam Noble headroom (equal-B: 8/56 = 1/7 coupling usage, T22).

0 IVA events — consistent with 4-beam H-anchor parent run observations. The H coupling geometry structurally disfavors the IVA corridor [TL_IVA, TL).
